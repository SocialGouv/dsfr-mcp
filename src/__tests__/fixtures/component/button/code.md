## Bouton

### Structure HTML

```html
<button class="fr-btn" title="Label bouton">Label bouton</button>
```

### Variantes

- `fr-btn--secondary` : bouton secondaire
- `fr-btn--tertiary` : bouton tertiaire
- `fr-btn--sm` : petite taille
- `fr-btn--lg` : grande taille
